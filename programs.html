Our Programs

Pearls Africa develops initiatives that expand access to coding education, artificial intelligence learning, and mentorship opportunities.

Ages 7–17 · Digital Skills · Flagship Program

GirlsCoding

Coding and digital skills for girls — teaching HTML, CSS, JavaScript, and Python through Summer Code Camps and AfterSchool programs. Building the next generation of female software developers.
